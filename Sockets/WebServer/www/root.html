You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /textAnalysis -- Enter text, returns word count and character count
  • /encrypt -- Enter text and value to shift in caesar cipher
  • /decrypt -- Enter text and value to shift in caesar cipher
